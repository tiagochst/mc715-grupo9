\section{Introdução}

\begin{frame}
  \frametitle{Como funciona?}
  \begin{itemize}
 \item Um cliente que quer ser o líder deve criar um znode efêmero e sequencial em "/ELECTION". 
\item O cliente cujo znode possuir o menor ID é nomeado o líder, enquanto os outros clientes esperam enquanto esse não caia. 
\item Caso o líder caia uma nova eleição é realizada.
\item  Para que a queda seja conhecida, é colocado um watcher sobre o znode do líder que é disparado quando ele cai. 
  \end{itemize}
\end{frame}
